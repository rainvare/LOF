# Carpeta de conversaciones históricas

Acá podés pegar conversaciones pasadas para que el agente LOF las analice
y enriquezca tu perfil cognitivo.

## Cómo usarla

1. Exportá o copiá conversaciones tuyas de cualquier chat (Claude, ChatGPT,
   foros, emails, lo que sea donde hayas escrito sobre temas que te interesen)

2. Guardá cada conversación como un archivo `.txt` o `.md` en esta carpeta.
   El nombre del archivo no importa.

3. Escribile al agente: `/lof analizar`

4. El agente va a leer todos los archivos, identificar patrones en cómo
   pensás y comunicás, y actualizar tu perfil cognitivo.

## Qué tipo de conversaciones funcionan mejor

- Conversaciones donde hayas hecho preguntas sobre temas que querías aprender
- Intercambios donde hayas pedido explicaciones o aclaraciones
- Cualquier texto donde hayas explicado algo con tus propias palabras
- Conversaciones técnicas o académicas donde se vea cómo procesás información

## Privacidad

Los archivos que ponés acá solo los lee el agente LOF localmente.
No se envían a ningún servidor externo — el análisis ocurre en tu
sesión de Claude Code.

## Formato sugerido (no obligatorio)

```
[Conversación con: ChatGPT / Claude / etc.]
[Fecha aproximada: mes año]

Usuario: [tu mensaje]
Asistente: [respuesta]
Usuario: [tu mensaje]
...
```

Podés pegar el texto como venga — el agente puede trabajar con
formatos informales.
